# Personal Finance & Wealth Audit Dashboard

## Overview
End-to-end personal finance analysis tracking income, 
expenses, savings rate and net worth across 12 months 
using Python, SQL, and Power BI.

## Problem Statement
Most people have no clear view of where their money 
goes month to month. This project builds a structured 
financial audit — identifying spending patterns, 
budget variances and savings trajectory.

## Tools Used
| Tool | Purpose |
|------|---------|
| Python (Pandas) | Data cleaning & enrichment |
| SQL (SQLite) | Analytical queries & validation |
| Power Query | ETL & data transformation |
| Power BI + DAX | Data modelling & dashboard |
| GitHub | Version control & documentation |

## Dataset
- 392 transactions across 12 months (2024)
- Categories: Income, Rent, Groceries, Transport, 
  Dining, Entertainment, Health, Shopping, 
  Savings, Investments
